## What Is KWANUS?

KWANUS is a **Financial Outcome Engine**. Not a credit repair app. Not a fintech tool. Not a feature stack.

Its single purpose: reduce financial chaos and guide users through the strongest sequence of financial actions.

## Core Outcome Loop

```
Repair -> Build -> Optimize -> Prepare -> Fund
```

Every feature, screen, engine, and API call must map to one of these five stages. If it doesn't, it gets deferred.
