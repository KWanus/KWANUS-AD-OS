# PHASE 1 UI / UX SPEC

## Visual direction
- dark mode default
- deep navy / black background
- cyan / teal accents
- rounded cards
- soft glow
- clean typography

## Pages
### Dashboard
Must show clear choices:
- Scan Products
- Scan Businesses
- Projects

### Scan Products
- one main action
- one results panel
- obvious next step

### Scan Businesses
- one URL input
- one scan button
- one results panel
- obvious save action

### Projects
- simple placeholder is acceptable in Phase 1
- must match visual system

## Interaction rules
- no clutter
- no long forms
- immediate feedback
- loading state required
- error state required
- success state required
