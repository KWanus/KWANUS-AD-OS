## How This Library Is Used

When a beginner selects Operator/Dropship mode and indicates they are starting from scratch,
the system picks one product from this library based on:
1. Niche preference (or auto-selects highest-confidence niche)
2. Starting budget ($20–50/day tier)
3. Platform preference (TikTok-first vs. Facebook-first)

The selected product then seeds the full asset generation pipeline:
ad hooks, scripts, landing page copy, email sequences, and execution checklist.

---

## Beginner-Friendliness Signal Framework

These are the signals the AI engine uses to rank products for complete beginners.
Each signal is weighted. Products in this library score 7+ out of 10.

### Green Signals (Beginner-Friendly)
| Signal | Why It Matters for Beginners |
|---|---|
| Demo completes in under 8 seconds | Beginner can film it with a phone. No editing skill needed. |
| Before/after is visually obvious | No copywriting needed — the product sells itself |
| Single use case | Easier to write ad copy. One problem, one product, one CTA. |
| Universal problem (affects 60%+ of adults) | Broad targeting works, no need for expert audience research |
| No sizing, no color matching | Zero return risk from wrong variants |
| Under 200g, ships in CJ Dropshipping Standard | Predictable fulfillment, no customs surprises |
| $35–55 price point | High enough for margin, low enough for impulse buy |
| Proven organic TikTok views (500K–5M) before paid | Demand is real, not manufactured |
| 5+ creative angles available | When one angle dies, pivot without changing product |
| Problem is embarrassing or private | Lower price sensitivity, higher purchase urgency |

### Red Signals (Requires Expertise)
| Signal | Why Beginners Should Avoid |
|---|---|
| Requires certification claims (FDA, CE) | Legal risk, ad account bans |
| Electronics with no clear certifications | PayPal/Stripe holds, chargebacks |
| Clothing or shoes | 25–40% return rate destroys margin |
| Already on every creator page | Window closed, CPMs skyrocketing |
| Seasonal product | 90-day window, then dead |
| Requires medical or scientific explanation | Ad copy too hard, compliance risk |
| $70+ price point | Beginner media buyers can't optimize to high CPP |
| Fragile in shipping | 1-star reviews sink new stores |
| Requires custom sizing/fitting | Returns, complaints, support overwhelm |
