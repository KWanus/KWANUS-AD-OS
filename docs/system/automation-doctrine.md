# Automation Doctrine

## Goal
The user should NOT have to think.

---

## What the System Must Do Automatically
- find/validate products
- generate ad hooks and scripts
- build funnel structure
- write email sequences
- create execution workflows
- score opportunities
- produce decision recommendations

---

## What the User Does
- review outputs
- fund ads
- execute steps in order
- approve or reject recommendations

---

## Automation Levels

### Level 1 — Assisted
System generates, user reviews, user executes.

### Level 2 — Semi-Automated (Phase 4+)
System generates + sequences tasks.
User approves then executes.

### Level 3 — Full Automation (Phase 7+)
System discovers, generates, and sequences.
User funds and confirms.

---

## Rule
Never remove the human from final approval.
The system decides. The human confirms.
