# Subscription Architecture

## Tier 1 — Starter ($)
Who: beginners, casual users
Access:
- limited scans per month
- basic link analysis
- simplified outputs (SaaS Mode only)
Not included:
- full strategy
- asset generation
- consultant-level reports

---

## Tier 2 — Operator ($$)
Who: affiliates, dropshippers, serious users
Access:
- full product analysis
- Operator Mode outputs
- ad hooks + scripts
- basic funnel structures
- execution checklists
Not included:
- advanced consultant packets
- exportable client reports

---

## Tier 3 — Consultant ($$$)
Who: freelancers, agencies, contractors
Access:
- full diagnosis engine
- full strategy engine
- Decision Packets (exportable, client-ready)
- deeper business audit outputs
- consulting mode

---

## Tier 4 — Pro / KWANUS Core ($$$$)
Who: power users, internal use
Access:
- everything
- all future features
- advanced workflows
- full system control
- no limits

---

## Output Filtering Rule
Same engine — filtered output per tier:
- Starter: summary + light recommendations
- Operator: hooks + ads + funnel + checklist
- Consultant: full strategy + formatted report + packet
- Pro: all layers, no filtering
