# HIMALAYA — Product Roadmap

## Phase 1: Intelligent Foundation Builder
**Status: Active**

Two paths: "Start from scratch" + "Improve existing business"

Delivers:
- Entry Engine (path selection, stage detection)
- Business Diagnosis Engine (analysis + scoring)
- Strategy Engine (prioritized action plans)
- Generation Engine (foundational assets)
- Dashboard (status, next actions, assets, roadmap)

See `PHASE_1_SPEC.md` for full details.

---

## Phase 2: Optimization + Operations
**Status: Queued**

Delivers:
- Optimization Engine (detect weaknesses, suggest + regenerate improvements)
- CRM pipeline management
- Basic automation builder
- Email/SMS flow management
- Before/after comparison tracking
- Analytics dashboard

---

## Phase 3: Business Model Specialization
**Status: Queued**

Delivers:
- Specialized flows per business model:
  - Local service: lead gen + booking + reviews
  - E-commerce: product pages + retention + upsells
  - Coach/consultant: authority funnel + booking + nurture
  - Agency: proposal system + client onboarding
- Model-specific templates, flows, and best practices
- Industry-aware generation

---

## Roadmap Rules

1. Phases execute in order. No skipping.
2. Each phase must be validated before the next begins.
3. Features not on this roadmap require explicit approval.
4. The sacred sequence is: Entry → Diagnosis → Strategy → Generation → Optimization
5. Every phase must strengthen this sequence, not add disconnected features.
6. If unsure whether a feature belongs, it gets deferred.
