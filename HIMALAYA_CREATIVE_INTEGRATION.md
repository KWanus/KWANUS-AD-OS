Navigate to `/creative-studio/himalaya` and the studio:

2. **Displays AI-generated ad creative ideas** for each platform
3. **Shows your business context** (headline, offer, niche)
4. **Provides marketing angles** Himalaya discovered

### **Step 3: Generate Professional Ads**

Click "Design Ad" on any creative and:

1. **Opens Canva-style editor** with your business data pre-loaded
2. **Hook text** auto-filled from Himalaya
3. **Visual style** recommendations applied
4. **Platform format** (1080x1080, 1080x1920) set correctly
5. **Drag-and-drop** to customize
6. **Export PNG/JPG** ready to upload

## 🔧 Technical Implementation

### **Data Flow:**

```typescript
// 1. Himalaya generates foundation
POST /api/himalaya/express
→ Creates analysis with foundation.adCreatives

// 2. Creative studio fetches projects
GET /api/himalaya/projects
→ Returns list of all Himalaya analyses

// 3. Load specific analysis
GET /api/analyses/{analysisId}
→ Returns full foundation data including adCreatives

// 4. Display creatives in UI
adCreatives.map(creative => {
  platform: "Meta",
  hook: "Your hook",
  visualStyle: "Professional split-screen",
  imagePrompt: "Full AI prompt for image generation"
})

// 5. User clicks "Design Ad"
/creative-studio/editor/{analysisId}?creative={JSON}
→ Editor pre-loads creative data
```

### **Key Code:**

**Loading Himalaya Projects:**
```typescript
const loadHimalayaProjects = async () => {
  const res = await fetch("/api/himalaya/projects");
  const data = await res.json();
  setProjects(data || []);
  if (data && data.length > 0) {
    setSelectedProject(data[0].id);
  }
};
```

**Loading Analysis Data:**
```typescript
const loadProjectAnalysis = async (projectId: string) => {
  const res = await fetch(`/api/analyses/${projectId}`);
  const data = await res.json();
  setAnalysis(data);
  // data.foundation.adCreatives → Array of AI-generated ideas
  // data.foundation.marketingAngles → Array of marketing hooks
};
```

**Displaying Creatives:**
```typescript
const adCreatives = analysis?.foundation?.adCreatives || [];

{adCreatives.map((creative, idx) => (
  <CreativeCard
    key={idx}
    creative={creative}
    analysisId={analysis.id}
  />
))}
```

## 📊 Data Structure

### **Himalaya Foundation → adCreatives:**

```typescript
type AdCreative = {
  platform: string;        // "Meta" | "TikTok" | "Google" | "Instagram"
  format: string;          // "image-ad" | "video-ad" | "carousel"
  hook: string;            // "Stop Back Pain with Ergonomic Design"
  visualStyle: string;     // "Before/after split screen, professional"
  imagePrompt: string;     // Full AI image generation prompt
  videoScript?: {
    duration: string;      // "30 seconds"
    hook: string;          // Opening hook
    problem: string;       // Problem statement
    solution: string;      // Your solution
    cta: string;           // Call to action
  };
};
```

### **Himalaya Foundation → marketingAngles:**

```typescript
type MarketingAngle = {
  hook: string;            // "Tired of back pain from your office chair?"
  angle: string;           // "Problem-solution with social proof"
  platform: string;        // "TikTok"
};
```
